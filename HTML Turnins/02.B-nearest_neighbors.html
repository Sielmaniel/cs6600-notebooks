Nearest Neighbor (NN)¶

Given and dataset $\boldsymbol{D} = [\boldsymbol{X}\ \ \boldsymbol{y}]$ and an unseen example $\boldsymbol{x}$, this classifier:

  • calculates the distances between $\boldsymbol{x}$ and every example in $\boldsymbol{D}$. Using the Euclidean distance, we have: $$ 
    distance(\boldsymbol{x^{(a)}}, \boldsymbol{x^{(b)}}) = \Vert\boldsymbol{x^{(a)}} - \boldsymbol{x^{(b)}}\Vert_2 = \sqrt{\sum_{i = 1}^{m} (x_i^{(a)} - x_i^{(b)})^2}
    $$
  • finds the example that is nearest to X and return its $y$

Here is a class implementing this classifier

In [2]:
class NearestNeighborClassifier:
    def __init__(self, dataset):
        self.ds = dataset
        
    def calc_distance(self, x, y):
        distance = 0.0
        for i in np.arange(len(x)):
            distance += (x[i] - y[i])**2
                
        return math.sqrt(distance)
    
    def predict(self, unseen):
        if unseen.ndim == 1:
            distances = np.full(self.ds.N, 0.0)
            for i, ex in enumerate(self.ds.inputs):
                distances[i] = self.calc_distance(unseen, ex)

            return self.ds.target[distances.argmin()]
        else:
            return np.array([self.predict(ex) for ex in unseen])

A more efficient implementation¶

The way class NearestNeighborClassifier calculated the distances can be improved by taking advantage of NumPy vectorization. Here is a shorter and improved version. Do you see the difference?

In [11]:
class NearestNeighborClassifier:
    def __init__(self, dataset):
        self.ds = dataset
    
    def predict(self, unseen):
        if unseen.ndim == 1:
            distances = np.sqrt(np.sum((self.ds.inputs-unseen)**2, axis=1))

            return self.ds.target[distances.argmin()]
        else:
            return np.array([self.predict(ex) for ex in unseen])

def metricCalc(confusionMatrix, index):
    #Set our initial values to 0
    tn, fp, fn, = 0, 0, 0
    #Grab the colum for the class we are examining, this is the data for tp and fp
    sm = cm[:, index-1:index]
    #tp will be the row that matches the index in our sm dataset
    tp = np.sum(sm[index-1:index])
    
    #Here we loop through each row in the original confusion matrix
    for i in range(sm.size):
        i = i + 1
        #if we are on a row that does not match the index
        if i != index:
            #Loop through the columns in cm
            for j in range(cm[:].size):
                j = j + 1
                #If our column does not match the index. This means we are not on the row or column that matches the index
                #Which means it is a tn value which we add to our tn count
                if j != index:
                    tn = tn + np.sum(cm[j-1:j, i-1:i])
            #If we are in a row that does not match the index we grab the value from our sm dataset to get a fn value
            fn = fn + np.sum(sm[i-1:i])
            #We go to the cm dataset and grab the values on the index row but not the index column. These are the fp
            fp = fp + np.sum(cm[index-1:index, i-1:i])
     
    
    #I did not create a 2x2 confusion matrix like the hint suggested
    #but the values I calculated could used to make one as follows:
    #[tp fp]
    #[fn tn]
    #This would be the 2x2 confusion matrix generated
    
    
    #Calculate sensitivity, specificity, precision, and f1        
    sens = tp/(tp+fn)
    spec = tn/(tn+fp)
    prec = tp/(tp+fp)
    f1 = 2 * (prec * sens) / (prec + sens)
    
    #return values
    return sens, spec, prec, f1
